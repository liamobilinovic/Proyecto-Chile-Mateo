Visualizador de Datos ChileMateo

ChileMateo es un visualizador datos públicos que tiene como objetivo compartir datos relevantes sobre nuestro país y que puedan ser de utilidad a la hora de elaborar políticas públicas y de informar a la ciudadania en general, poniendo foco en la transparencia e inclusividad de esta herramienta.

Visualizadores

Visualizador Chile

Este visualizador presenta datos de rendimiento escolar y relacionados de las comunas de todo Chile. El visualizador actualmente abarca los años desde 2018 a 2023.

Este fue realizado a través del paquete de R "Shiny". En nuestro GitHub puede encontrar el código y bases de datos que generaron este visualizador

Visualizador Gran Santiago

Este visualizador presenta datos de rendimiento escolar y relacionados de las comunas del Gran Santiago. El visualizador actualmente abarca los años desde 2018 a 2023.

Este fue realizado a través del paquete de R "Shiny". En nuestro GitHub puede encontrar el código y bases de datos que generaron este visualizador

Este proyecto fue realizado por estudiantes de la Pontificia Universidad Católica de Chile participando en el proyecto Transparenta 2024.
